# Medical-Chatbot-project


# STEPS:

Clone the repository
```bash
Project repo: https://github.com/Tinonturja/Bangla-Medical-Chatbot-Project/tree/main
```

# STEP 01: Create a conda environment after opening the repository

```bash
conda create -n banglamedicalchatbot python=3.10 -y
```

```bash
conda activate banglamedicalchatbot
```


### STEP 02- install the requirements
```bash
pip install -r requirements.txt
```
